The Project Name is Part_One
This project consists of various initial HTML coding pages used to create a structured layout of my website of D & A Planning co.
The HTML pages in this project includes
1. Home Page (index.html) - all information that leads to different pages such as an overview of the website
2. About Page (about.html) - states the mission, vision and the story of how D & A planning co came about
3. Services Page (services.html) - states all 3 service packages that the company offers such as booking consultations
4. Enquiry Page (enquiry.html) - allows clients to track their plan to see its progress
5. Contact Page (contact.html) - gives contact information as well as live location for clients to effectively contact if they have concerns or problems.

Styled CSS page was added to create a structured layout to organise all revelant information about D & A Planning co.
